A simplified Flappy Bird game built with Python and pygame.
Control the bird to fly through pipes without hitting them.

Features
Flappy Bird style gameplay with gravity and velocity.
Randomly generated pipes with variable gaps.
Bird flap animation using two wing images.
Collision detection for ceiling, floor, and pipes.
Reset the game automatically on collision.

Controls
SPACE – Flap the bird
hold SPACE – Sprint the bird
Close window – Quit game
